== Description ==

Plugin Jeedom pour gérer un Spark Core https://www.particle.io/

De plus, le plugin permet la gestion de zones de radiateur grâce aux fil pilotes, la récupération de téléinformations, la récupération de la température grâce à un programme http://github.com/projetBTSMaisonIntelligente/
