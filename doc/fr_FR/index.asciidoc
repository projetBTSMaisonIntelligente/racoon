= Racoon =

== Description ==

Plugin Jeedom pour la gestion des radiateurs dans sa maison grâce à des fils pilote contrôlés par un Spark Core.

=== Gestion d'une zone ===

*Page de configuration*

image::../images/pageConfig.PNG[]

*Device ID du Spark Core : Identifiant de l'appareil accessible sur https://build.particle.io/build/[Particle Build].

*Token d'accès du Spark Core : Clé d'accès à l'appareil accessible sur https://build.particle.io/build/[Particle Build].

image::../images/idSparkCore.png[]

image::../images/accessTokenSparkCore.png[]

*Tableau des commandes*

image::../images/pageGestionZone.png[]

*Arrêt : Envoi d'un ordre arrêt dans la zone sélectionnée.

*Confort : Envoi d'un ordre confort dans la zone sélectionnée.

*Hors-gel : Envoi d'un ordre hors-gel dans la zone sélectionnée.

*Eco : Envoi d'un ordre economique dans la zone sélectionnée.

*Statut : Permet de récupérer le statut de la zone sélectionnée.
